# Focus Timer

Single-page Pomodoro web app with:

- Mobile-first responsive UI
- Binaural beat profiles (focus, creativity, relaxation)
- Synth pad masking during focus
- End-of-focus red-noise cue with a 20-second fade-in and filter movement
- Manual transition button into break
- Musical break cues with ocean-like ambient noise and harmonic return swell
